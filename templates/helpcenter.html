Getting Started File Requirements Credits & Billing Troubleshooting

Getting started

  1. Go to Dashboard and choose the processing flow you need.
  2. Use Lecture Notes when you have slides and audio for the same lecture.
  3. Use Slides Extraction when you only need clean text from slides.
  4. Use Interview Transcription when you only have recorded audio.
  5. Save the generated study pack, then review it in Study Library.

File requirements

  • Slides: PDF or PPTX, up to 50 MB.
  • Audio: MP3, M4A, WAV, AAC, OGG, or FLAC, up to 500 MB.
  • Lecture Notes currently requires both slides and audio. An LMS import can serve as the audio source.
  • Slides Extraction requires slides only.
  • Interview Transcription requires audio only.

Credits and billing

  • Lecture Notes uses lecture credits.
  • Slides Extraction uses text extraction credits.
  • Interview Transcription uses interview credits. Optional extras can add text extraction credit usage.
  • Buy credits from Buy Credits. Checkout is handled by Stripe.
  • If a run fails because of an internal processing error, the app may refund credits based on current refund rules.

Batch processing

  • Open Batch Processing when you need to process many rows in one submission.
  • Pick the batch type on the page tabs, not in multiple places in the sidebar.
  • Use batch mode when you want one queue, one progress view, and one export ZIP.
  • Check Batch status for recent batch status.

Study Library and planning

  • Use Study Library to organize packs into folders.
  • Use Planning & Progress to set a daily review goal and add exam dates per folder.
  • Use Calendar to track planned sessions that sync with your account.
  • Browser reminders only fire while an active calendar tab is open on that device.
  • Progress and due counts improve after you start studying saved packs.

Troubleshooting

  • If a button looks disabled, check that you are signed in and have the required files uploaded.
  • If credits look wrong, refresh once, then open the account menu or Buy Credits page to verify the balance.
  • If a lecture video import fails, retry with the normal lecture page first, then try the direct media playlist URL and confirm it still plays in your browser session.
  • If study pages look empty, upload a first lecture or create a folder in Study Library to unlock the next steps.
